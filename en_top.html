ORIZIN Agent is an open source AI assistant.


Open Source

ORIZIN Agent is a simple AI assistant which is published on GitHub under MIT license.It works on Linux.If you want to know other conditions of it, please check the Wiki.
Wiki


Off-line

ORIZIN Agent's basic functions can work off-line.If ORIZIN Agent can access the Internet, it can read news for you.


Versatility

ORIZIN Agent uses tkinter which is standard library of Python3 for graphic, so you have to install little libraries.


Abundant Functions

ORIZIN Agent has a lot of functions.For example, saying tongue twister, reading news etc.


Easy to Add Functions

You can change the words ORIZIN Agent can understand and its response easily.ORIZIN Agent's computer programs are simple, so if you want to add functions, you can do so easily (If you have written Python programs before).
Detail


NOTICE:ORIZIN Agent corresponds to only Japanese.But if you will improve the computer programs, I think ORIZIN Agent will be able to understand other languages.
