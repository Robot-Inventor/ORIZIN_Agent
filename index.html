ORIZIN Agent は、オープンソースの AI アシスタントです。


オープンソース

ORIZIN Agent は、MIT ライセンスの下に GitHub 上で公開されている簡易的な AI アシスタントです。Linux 上で動作します。その他の動作環境は、Wiki をご覧ください。
Wiki


オフライン

基本機能は全てオフラインで動作します。また、インターネット接続があれば、ニュースの読み上げにも対応します。


互換性

GUI に Python 標準ライブラリの tkinter を使用しています。その為、新たにインストールする必要があるライブラリ等は最小限に抑えられています。


豊富な機能

早口言葉やジャンケン、ニュース読み上げ、音楽の再生等にも対応します。


簡単な機能拡張

ORIZIN Agent が認識できる言葉とそれに対する応答は、テキストファイルを編集するだけで自分好みに変更できます。また、ORIZIN Agent のプログラムは単純なので、機能の追加も比較的簡単に行えます。もし、あなたが Python を使ったことがあるなら、それは驚くほど簡単でしょう。
