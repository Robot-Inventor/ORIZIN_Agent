ORIZIN Agentについて

ORIZIN Agent は、2019年5月(推定)に開発が始まったMIT ライセンスの下に GitHub 上で公開されている簡易的な AI アシスタントです。

キャッチコピーについて

ORIZIN Agentのキャッチコピーは「いつもあなたと共に。オープンな相棒」、英語版では「Always with you.Your open partner」です。「オープンな相棒」と「open partner」は、ORIZIN Agentがオープンソースで、誰でも使えるということを表しています。また、日本語版、英語版ともに一番最後に句点やピリオドが無いことには、終わりが無いつまりORIZIN Agentが進化し続けるという意味が込められています。ORIZIN Agentには学習する機能は付いていませんが、辞書を編集することで挙動を変えることができ、誰かが辞書を編集し続ける限りORIZIN Agentは進化し続けるのです。

ORIZIN Agentのロゴについて

ORIZIN Agentのロゴは、次のようになっています。フォントは「IPA Pゴシック」を使用しています。IPA Pゴシックは、IPA（独立行政法人　情報処理推進機構）が作成しているフォントです。また、配色については、「ORIZIN Agent」という文字はオレンジ、その他のキャッチコピーや、ORIZIN Agent Projectロゴの「project」という文字は、グレーです。もっと厳密にいえば、オレンジもグレーもPaint.netのプリセットカラーで、オレンジが#FF6A00、グレーが#404040です。
